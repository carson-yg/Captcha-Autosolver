Captcha Solver

CAPTCHA stands for Completely Automated Public Turing test and is used to tell computers and humans apart. It is used by some web sites to ask users' input and restrict the usage of computer bots.

There are many types of CAPTCHAs and we are going to use the most simple one - A text image to demonstrate how to solve it. There are many libraries available to help solve the text in CAPTCHA, like Optical Character Recognition (OCR), Google Vision API, or Amazon Textract. In this project I plan to use the images generated by captchas.net and feed them to Google Vision API

I used Tomcat with a Spring boot framework and jQuery. When a user accesses the web page, a request is sent to captchas.net to get the image then saved in the folder of the server. A request to Google Version API will help resolve the text in Captchas.
